# 字段说明

## 打分数据字段

| 字段名 | 类型   | 说明                                      |
| ------ | ------ | ----------------------------------------- |
| date   | string | 数据日期，格式 YYYY-MM-DD                 |
| score  | number | 打分结果，范围 0-100                      |
| cs     | number | CS值（收盘价与EMA20乖离率，衡量短期强度） |
| rsi    | number | RSI值（Relative Strength Index）          |

## 分数含义

### 超买超卖判断

| 分数范围 | 状态 | 投资含义                 |
| -------- | ---- | ------------------------ |
| 0-20     | 超卖 | 市场过度悲观，可考虑抄底 |
| 20-40    | 偏低 | 相对低估区间             |
| 40-60    | 中性 | 正常波动区间             |
| 60-80    | 偏高 | 相对高估区间             |
| 80-100   | 超买 | 市场过度乐观，需考虑止盈 |

## 名词解释

### 打分（Score）

基于多种技术指标综合计算的量化分数，用于判断当前市场的超买超卖状态。分数越高表示越接近超买，分数越低表示越接近超卖。

### CS值（Close-EMA20 偏离度）

收盘价与 EMA20（20日指数移动平均线）的乖离率，用于衡量短期强度。

**计算公式**：`CS = (收盘价 - EMA20) / EMA20 * 100`

**含义解读**：

- CS > 0：价格在 EMA20 上方，短期偏强
- CS < 0：价格在 EMA20 下方，短期偏弱
- CS 越大：短期强度越强，但需警惕过热
- CS 越小：短期强度越弱，但可能存在反弹机会

### RSI值（Relative Strength Index）

相对强弱指数，衡量价格变动的速度和变化，范围 0-100。

- RSI > 70：超买信号
- RSI < 30：超卖信号

## 趋势分析

### 上升趋势

- 分数持续上升
- 低点逐步抬高
- 可能接近超买

### 下降趋势

- 分数持续下降
- 高点逐步降低
- 可能接近超卖

### 震荡趋势

- 分数在一定区间波动
- 无明显方向
- 需等待突破信号

## 使用注意事项

1. **数据滞后性**：打分数据基于历史价格计算，具有滞后性
2. **极端行情**：在极端行情下，打分可能持续超买或超卖
3. **综合判断**：建议结合多个指标和市场环境综合判断
4. **周期选择**：不同分析周期可能得出不同结论
