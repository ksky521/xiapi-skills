# CS 指标说明

## 什么是 CS 指标

CS（Close-EMA20 偏离度）是收盘价与 EMA20（20日指数移动平均线）的乖离率，用于衡量短期强度。

## CS 指标计算

**计算公式**：`CS = (收盘价 - EMA20) / EMA20 * 100`

**计算逻辑**：
- EMA20 是 20 日指数移动平均线，对近期价格赋予更高权重
- CS 反映当前价格相对于 EMA20 的偏离程度
- 正值表示价格在均线上方，负值表示价格在均线下方

## CS 阈值解读

### 入场判断

| CS 范围 | 状态   | 操作建议       |
| ------- | ------ | -------------- |
| > 15    | 过热   | 不追高         |
| 5 ~ 15  | 适中   | 可考虑入场     |
| < 5     | 偏低   | 谨慎观望       |

### 持仓判断

| CS 范围 | 状态     | 操作建议     |
| ------- | -------- | ------------ |
| > 0     | 正常     | 无需担忧     |
| -5 ~ 0  | 底部     | 建议坚守     |
| < -5    | 深度底部 | 建议离场     |

## CS 指标的核心意义

### 衡量短期强度

- **CS > 0**：价格在 EMA20 上方，短期偏强，多头力量占优
- **CS < 0**：价格在 EMA20 下方，短期偏弱，空头力量占优
- **CS 绝对值越大**：短期强度越强，但需警惕过热或超卖

### 均线偏离判断

- **CS 过大（如 > 15）**：价格严重偏离均线，可能存在回调风险
- **CS 过小（如 < -15）**：价格严重低于均线，可能存在反弹机会

## 使用注意事项

1. **滞后性**：CS 指标具有滞后性，可能错过最佳买卖时机
2. **极端行情**：在极端行情下，CS 可能持续高位或低位
3. **结合趋势**：建议结合趋势分析，不单独使用
4. **多周期验证**：建议多周期验证信号

## 板块分类标准

### 领涨板块

- CS > 5
- 可能成为领涨龙头
- 需关注是否过热（CS > 15）

### 上升板块

- 连续多日 CS 上升
- 趋势向上
- 可关注回调机会

### 反转板块

- 处于底部区域（CS < 0）
- CS 连续 2 次上涨
- 可能存在反转机会
