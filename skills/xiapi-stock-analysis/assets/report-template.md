# 个股分析报告模板

> **Agent 使用说明**：
> - 严格按照本模板格式输出，不得随意增删章节、改变顺序或改变表格结构。
> - 标注「必填」的节每次必须输出；「条件必填」满足触发条件时必须输出；「可选」有实质内容时才输出，无内容则整节省略。
> - 结论先行：每节开头先给结论句，再给数据支撑。
> - 禁止无依据结论：所有判断必须引用具体字段值。
> - 金额统一换算为万元，保留整数（如 70085269 元 → 700万元）。
> - 多股对比时必须统一口径（时间窗/报表期/指标定义），财务表格必须标注报告期。
