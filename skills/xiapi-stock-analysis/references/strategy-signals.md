# 策略逻辑信号表

本文档定义个股分析的核心信号组合与判断规则，优先使用 `daxiapi` 字段。

---

## 核心信号组合

| 信号组合 | 含义 | 判断 |
|---------|------|------|
| `zdf5 > zdf10 > zdf20` 且 `zdf > 0` | 短中期动量共振向上 | ✅ 积极 |
| `zdf5 < zdf10 < zdf20` 且 `zdf < 0` | 短中期动量共振转弱 | ⚠️ 警惕 |
| `CS`（或 `bais`）在 `0~15` 且 `IBS >= 70` | 强势但未明显过热，收盘质量较好 | ✅ 积极 |
| `CS`（或 `bais`）`> 15` | 相对20EMA偏离过大，追高风险上升 | 🟡 关注 |
| `CS`（或 `bais`）`< -5` 且 `IBS <= 30` | 跌破20EMA且弱收盘，短期承压 | ⚠️ 警惕 |
| `SCTR >= 70` 且 `zdf20 > 0` | 相对强度靠前且中期趋势仍在 | ✅ 积极 |
| `SCTR < 50` 且 `zdf20 < 0` | 相对强度偏弱且中期趋势走差 | ⚠️ 警惕 |
| 命中 `stock pattern trendUp` 或 `crossMa50` | 趋势信号转强（形态侧） | ✅ 积极 |
| 命中 `stock pattern vcp` 且 `IBS >= 60` | 波动收缩后有放量突破预期 | 🟡 关注 |
| `zdf20 < 0` + `CS < -5` + `SCTR < 50` | 趋势、强度、位置三项共振转弱 | 🔴 高风险 |

## 字段说明

### zdf 系列（涨跌幅）

| 字段 | 含义 | 用途 |
|------|------|------|
| `zdf` | 当日涨跌幅 | 判断当日强弱 |
| `zdf5` | 5日累计涨跌幅 | 短期动量 |
| `zdf10` | 10日累计涨跌幅 | 中短期动量 |
| `zdf20` | 20日累计涨跌幅 | 中期动量 |

### CS / bais（收盘价相对均线强度）

| 取值范围 | 含义 | 操作建议 |
|---------|------|---------|
| `0 ~ 15` | 健康偏强区间 | 正常持有或关注 |
| `> 15` | 短期偏离过大 | 警惕追高风险 |
| `< -5` | 明显跌破20EMA | 关注风险 |

### IBS（内部K线强度）

| 取值范围 | 含义 |
|---------|------|
| 接近 100 | 收盘接近当日高点，K线质量好 |
| 接近 0 | 收盘接近当日低点，K线质量差 |
| `>= 70` | 高位收盘，强势 |
| `<= 30` | 低位收盘，弱势 |

### SCTR（市场相对强弱排名）

| 取值范围 | 含义 |
|---------|------|
| `>= 70` | 相对强度靠前，强势股属性 |
| `50 ~ 70` | 中等强度 |
| `< 50` | 相对强度偏弱 |

## 组合信号实战示例

### 示例 1：积极信号共振

> **标的**：贵州茅台（600519），假设某日 `stock info` 返回如下快照：

| 字段 | 值 | 信号 |
|------|-----|------|
| `zdf` | +1.2% | 当日收涨 |
| `zdf5` | +3.8% | 短期动量向上 |
| `zdf10` | +5.1% | 中短期动量向上 |
| `zdf20` | +8.2% | 中期动量向上 |
| `CS`（bais） | 8.5 | 健康偏强区间 |
| `IBS` | 82 | 高位收盘，K线质量好 |
| `SCTR` | 78 | 相对强度靠前 |

**命中信号组合**：
- `zdf5 > zdf10 > zdf20` 且 `zdf > 0` → ✅ 短中期动量共振向上
- `CS` 在 `0~15` 且 `IBS >= 70` → ✅ 强势但未明显过热
- `SCTR >= 70` 且 `zdf20 > 0` → ✅ 相对强度靠前且中期趋势仍在

**综合判断**：三组积极信号共振，技术面偏强。需结合基本面和资金面做最终评级。

---

### 示例 2：高风险信号共振

> **标的**：某 ST 股票（假设代码 XXXXXX），假设某日 `stock info` 返回如下快照：

| 字段 | 值 | 信号 |
|------|-----|------|
| `zdf` | -2.5% | 当日下跌 |
| `zdf5` | -6.3% | 短期动量向下 |
| `zdf10` | -11.2% | 中短期动量向下 |
| `zdf20` | -18.7% | 中期动量向下 |
| `CS`（bais） | -12.3 | 明显跌破20EMA |
| `IBS` | 15 | 低位收盘，K线质量差 |
| `SCTR` | 22 | 相对强度极弱 |

**命中信号组合**：
- `zdf5 < zdf10 < zdf20` 且 `zdf < 0` → ⚠️ 短中期动量共振转弱
- `CS < -5` 且 `IBS <= 30` → ⚠️ 跌破20EMA且弱收盘
- `SCTR < 50` 且 `zdf20 < 0` → ⚠️ 相对强度偏弱且中期趋势走差
- `zdf20 < 0` + `CS < -5` + `SCTR < 50` → 🔴 高风险

**综合判断**：趋势、强度、位置三项共振转弱，触发高风险规则，建议在报告中明确输出"不建议持有"并附可验证原因。

---

## 输出要求

- 优先基于 `stock info` 与 `stock pattern` 的真实字段填充
- 仅当需要扩展指标（如 MA、MACD、RSI、KDJ）时再用 `kline` 复算
- 若个别字段暂不可得，需明确标注"数据缺失，未纳入判断"

---

## 高风险信号组合

当出现以下组合时，需在报告中明确输出"不建议持有"：

```
zdf20 < 0 + CS < -5 + SCTR < 50
```

**含义**：趋势、强度、位置三项共振转弱

**必须配合**：至少 3 条可验证原因，例如：
- 盈利持续下滑
- 中长期均线走坏、关键支撑失守
- IBS 低位弱收盘、热度下降、量价背离
- 行业景气下行
- 重大治理或合规风险
