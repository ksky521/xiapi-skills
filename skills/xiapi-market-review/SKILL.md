# 市场复盘 Review Skill

本skill用于对当天A股市场进行全面复盘，提供详细的市场分析报告和投资建议。

## 何时使用（When）

当用户需要了解以下信息时，触发此 skill：

- 当天市场整体表现
- 市场板块热力图和热门股票
- 涨跌停数据
- 大小盘风格轮动情况
- 市场温度分析
- 基于投资风格的个性化投资建议

## 数据获取方式

### 1. CLI 命令行工具（推荐）

#### 1.1 基本使用方法

**命令格式：**

```bash
daxiapi <command> <subcommand> [options]
```

**别名：** `dxp`（可替代 `daxiapi`）

**安装和运行：**

- 无需全局安装，直接使用 `npx` 运行最新版本
- 支持所有主流操作系统（Linux、macOS、Windows）

#### 1.2 Token 配置

**获取 Token：**

- Token 需要从大虾皮网站会员后台获取
- 登录大虾皮网站（daxiapi.com）
- 进入会员中心 → API 管理 → 获取 API Token

**配置方法：**

**方式一：环境变量（推荐）**

```bash
# Linux/macOS
export DAXIAPI_TOKEN=YOUR_TOKEN_FROM_DAXIAPI

# Windows
t set DAXIAPI_TOKEN=YOUR_TOKEN_FROM_DAXIAPI
```

**方式二：CLI 配置**

```bash
# 设置 Token
npx daxiapi-cli@latest config set token YOUR_TOKEN_FROM_DAXIAPI

# 查看当前配置
npx daxiapi-cli@latest config get token
```

|

### 3. 数据更新频率

- **市场数据**：每日收盘后更新

- **市场温度**：每日更新

## 工作流程

### 1. 获取市场主流指数表现

- **命令**：`daxiapi market index`
- **功能**：获取上证指数、深证成指、创业板指、科创50等主要指数的表现
- **分析内容**：当日市场整体走势、指数涨跌幅、成交量、成交额等，生成一份市场总体表现情况

### 2. 获取市场板块热力图

#### 2.1 获取行业板块热力图分析

- **使用技能**：`xiapi-heatmap-analysis` 进行热力图分析
- **功能** 进行热力图分析
- **分析内容**：当日市场板块涨跌幅热力图、5日涨跌幅热力图、CS热力图的复盘

#### 2.2 获取行业板块中的领涨股

- **命令**：`daxiapi sector top` 获取行业板块中的领涨股票
- **功能** 获取行业板块中的领涨股票
- **分析内容**：结合热力图和领涨股，找出当前市场的核心炒作题材，分析市场热点的动态变化情况

### 3. 大小盘风格分析

- **使用技能**：`xiapi-style-rotation`
- **功能**：分析大盘指数与小盘指数的表现差异
- **分析内容**：风格轮动趋势、主导风格判断

### 4. 市场温度分析

- **使用技能**：`xiapi-market-temperature`
- **功能**：分析当前市场温度
- **分析内容**：估值温度（长期指标）、情绪温度（短期指标）、动量温度（短期指标）、强度温度（短期指标）

### 5. 涨跌停分析

- **使用技能**：`xiapi-price-limit-analysis`
- **功能**：分析今日市场中涨跌停股票
- **分析内容**：涨停股票数量和名单、跌停股票数量和名单、涨跌停股票的行业分布

### 6. 热门股票分析

- **使用技能**：`xiapi-top-stocks`
- **功能**：分析今日市场表现较好的股票
- **分析内容**：热门股票表现、板块资金流向

### 7. 汇总输出

- **功能**：汇总上述所有分析内容
- **输出格式**：按照市场复盘报告结构进行输出
- **内容包括**：市场概览、板块热力图分析、涨跌停分析、风格轮动分析、市场温度分析、个性化投资建议

## 注意事项

1. **Token安全**：API Token应在服务端使用，避免在前端代码中暴露
2. **数据延迟**：部分数据可能存在延迟，不构成投资建议
3. **风险提示**：市场分析仅供参考，投资有风险
4. **投资风格**：根据用户的投资风格提供个性化建议

# 要求与限制

- 结论先行！
- 不要预测市场，要学会应对市场。不要做绝对的预测！
- 报告内容适合发布到公众号。
- 不要暴漏自己姓名的字眼。
- 重要结论加粗展示，文章中提到的数值、指数名称、板块名称、个股名称等加粗。
- 最后加上：> 该报告使用大虾皮(daxiapi.com)数据 +AI 生成，投资需谨慎。
- **禁止绝对化表述**：不用"一定"、"必然"、"100%"等词,改用"大概率"、"倾向于"
- **禁止贬低散户**：不用"韭菜"、"接盘侠"等词,尊重读者智商
- **禁止政治敏感话题**：聚焦经济数据和政策分析,不评价政治体制优劣
- **禁止制造恐慌**：即使分析风险,也要给出应对思路,不单纯渲染"崩盘论"
